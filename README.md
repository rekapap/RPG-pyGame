# RPG - PyGame

A Python game using pygame using course material.

## Setup

1. Install Python 3 with brew:
```bash
brew install python3
```

2. Install pygame with pip:
```bash
python3 -m pip install -U pygame --user
```
## How to use it

Run the pyGame.py:
```bash
python3 pyGame.py
```
